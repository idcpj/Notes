## ajax 提交
```js
$("form").submit(function(e){
        e.preventDefault();
        var data = $("form").serializeArray()
        var url = $("form").attr('action');

        $.post(url,data,function(result){
            if (result.code == 200){
                alert(result.message);
            }else{
                alert(result.message);
            }
        },'JSON');

    });
```
