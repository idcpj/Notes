## 技巧
|代码|效果|
|---|---|
|`main`| `public static void main(String[] args) {    }`|
|`ps`|`private String `|
|`pi`|`private int `|
|`print` |`System.out.println()`|
|`"hello word".sout`  | `System.out.println("hello word");`|
|`values.iter`|`for (String value : values) {        }`|
|`100.fori`|`for (int i = 0; i < 100; i++) { }`|
