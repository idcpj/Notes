## es6 语法格式错误
`.eslintrc.js`中添加关闭的即可
如 :`no-multiple-empty-lines` 错误
```
rules: {
    'no-multiple-empty-lines':0
}
```

常用可关闭的
```
rules: {
    "indent": 0,
    'space-before-function-paren':0,
    'no-multiple-empty-lines':0,
    'space-before-blocks':0,
}
```
重启`npm run dev`
