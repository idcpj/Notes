## 使用`stylus` 编辑
> [参考文档]()
```
npm install stylus --save-dev
npm install stylus-loader --save-dev
```
在`..vue`文件中使用
```
<style scoped lang="stylus" type="stylesheet/stylus">
    #app
        .tab
            display:flex
            color:red

</style>
````
