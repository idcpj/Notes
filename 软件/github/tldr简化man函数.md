## 安装
```
//node 安装
npm install -g tldr
//bash 安装
mkdir -p ~/bin
curl -o ~/bin/tldr https://raw.githubusercontent.com/raylee/tldr/master/tldr
chmod +x ~/bin/tldr
export PATH="$PATH:~/bin"
//go
go get github.com/pranavraja/tldr
```

## 使用
```
tldr zar
```
