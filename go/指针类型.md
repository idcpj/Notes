# var 与 new 区别
```go
p := new(person)
err := xml.Unmarshal(str, p)
```
同
```go
var p person
err := xml.Unmarshal(str, &p)
```
> 对于new 相当于分配引用类型的内存地址,并且赋值,而 var 只分配内存,没有赋值

```
var c *int
*c = 10
fmt.Println(*c)  //会报错

/*====*/

b := new(int)
*b = 10
fmt.Println(*b)  //10 
```
