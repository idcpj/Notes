## 入门
` curl www.sina.com`
## 输出到文件
获取网站源码
`curl -o ~/demo.html www.baodu.com`
## 常用参数
```
-i        显示响应头信息与内容
-I        只显示响应头信息
-b"name=xxx"   携带 cookie 信息
_H "Content-Type:application/json"  增加头信息
-d       post 的 body
--referer http://www.example.com        添加referer来源
```

## 常用场景
### post 请求
`curl -d '{"user_name":"idcpj","pwd":"12345"}' http://127.0.0.1:8000/user`
### 携带头信息
`curl -H "Content-Type:application/json"`
### 添加 referer
`curl --referer "http://www.example.com" `
### 携带 cookie
`curl -b "name=xxx"`
### 文件上传
`curl --form upload=@localfilename --form press=OK [URL]`
